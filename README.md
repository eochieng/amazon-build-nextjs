# Description

Amazon clone featuring:

- Next.js
- Tailwind CSS
- Redux
- Firebase
- Stripe
- NextAuth
- Responsive Design
- fakestoreapi.com (for pulling the product data)
- Heroicons (for icons)

# Installation Steps

1. Prereq: Use node 14 (`nvm use 14`)

2. `npm install --global yarn`

3. `yarn install`

4. `yarn run dev`
